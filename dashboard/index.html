שאלות ותשובות מהשטח — מתעדכן אחרי כל שיחה עם יזם
מה זה פרוטוקול HANDOFF?
שפה רשמית להעברת משימה בין אופיסרים — מי שולח, מי מקבל, ומה מבקשים. מונעת "נפילה בין כיסאות".
CEO · 2026-04-05
מה זה Seniority Score?
ציון ניסיון שכל אופיסר צובר עם כל פרויקט. כמו רופא ותיק מול מתמחה — יותר ניסיון = ייעוץ איכותי יותר.
CEO · 2026-04-05
מה זה MOAT (חפיר)?
מה שמונע ממתחרים להעתיק אותך. ה-MOAT של HypeSHow: ככל שיותר לקוחות עובדים איתנו, האופיסרים לומדים יותר — יתרון שמתחרה חדש לא יכול לקנות.
CEO · 2026-04-05
האם Backend חייב לחכות ל-UX?
לא — הם יכולים לרוץ במקביל. אבל לפני שה-CTO כותב שורת קוד, ה-CPO צריך להגדיר מה המשתמש עושה במוצר. שבועיים חוסכים חודש.
CEO · 2026-04-05
מה ההבדל בין Node.js ל-Python ל-Backend?
Node.js = שפה אחת לכל המערכת, מהיר לבנייה, מתאים ל-MVP. Python = חזק ל-AI/ML, אבל מוסיף מורכבות. ההחלטה: Node.js עכשיו.
CTO · 2026-04-05
מה זה NPS?
Net Promoter Score — "כמה סביר שתמליץ עלינו?" סקלה 0–10. מעל 9 = Promoter. מתחת ל-7 = Detractor. מדד הזהב של SaaS.
CEO · 2026-04-05
מה זה CTA ולמה יש First Login vs Daily Login?
CTA = Call To Action. First Login → "Meet your team" (רגע אמוציונלי). Daily Login → "Ask your team" (חזרה לעבודה).
CPO · 2026-04-06
למה CSS overflow:hidden לא מצליח לחתוך קשתות בדיוק?
כי CSS clip מחושב לפי גבולות ה-container. SVG clipPath נותן שליטה מתמטית מדויקת — אפשר לחתוך בדיוק ב-x=25.
CPO · 2026-04-06
מה פלטת המיתוג של HypeSHow?
שחור (#1c1c1e) + קורל (#e8623a). שחור = מקצוענות. קורל = יצירתיות ואנושיות. "WeWork של האונליין".
CEO+CPO · 2026-04-06
מה זה Model Routing ולמה זה חוסך כסף?
כמו מסעדה עם שני שפים: שאלות פשוטות → Haiku (זול), מורכבות → Sonnet. חלוקה: 80/15/5. הלקוח לא יודע מי ענה. חיסכון ~60%.
CFO+CEO · 2026-04-06
מה זה Prompt Caching?
System prompt (~2,500 טוקן) נשמר ב-Anthropic ומחויב ב-10% בלבד מהפעם השנייה. flag אחד בקוד = חיסכון 90%.
CTO · 2026-04-06
מה זה Context Management / Sliding Window?
ה-AI קורא מחדש את כל ההיסטוריה בכל הודעה. Sliding Window = שומרים 20 הודעות אחרונות + סיכום. "המזכירה שמסכמת". חיסכון ~40%.
CEO · 2026-04-06
מה זה Soft Throttling ולמה עדיף על Hard Block?
עיכוב הדרגתי: 70%→12שנ, 90%→30שנ, 100%→60שנ, 110%→עצירה. הלקוח מרגיש "המערכת עמוסה" — לא שנחסם. כל ההודעות מגיעות מהאופיסר, לא כשגיאת מערכת.
CFO+CISO · 2026-04-06
למה Supabase ולא Vercel Postgres?
Supabase = PostgreSQL + Auth מובנה + Row Level Security + Real-time + Connection Pooling. Free tier עד ~20 משתמשים. RLS מגן על נתונים ברמת ה-DB — גם אם יש bug בקוד.
CTO+CISO · 2026-04-06
מה זה Design Handoff?
הרגע שדניאל מעביר את העיצוב לעודד — לא "תראה קובץ" אלא מסירה מסודרת: צבעים, ריווחים, וכל state אפשרי (hover, active, error, empty). בלי זה המפתח מנחש.
CEO · 2026-04-06
מה זה Dev-Ready Spec?
המסמך שיוצא מה-Handoff. עונה על כל שאלה שעודד עשוי לשאול: אילו נתונים מגיעים מה-API? מה ה-empty state? מה הגודל ב-mobile? Spec טוב = קוד בלי עצירות.
